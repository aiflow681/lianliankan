【游戏名称】
连连看

【游戏介绍】
这是一款H5原生开发的经典益智消除类小游戏。游戏中，你需要在规定时间内找出所有相同的图块并将它们连接消除。游戏采用可爱的水果emoji作为图块，画面简洁清新，操作简单但极具挑战性。共有9个关卡，难度逐渐递增！

【玩法说明】
游戏中，棋盘上分布着各种水果图块，你需要找出相同的图块并连接消除它们。

障碍：
1. 每个关卡都有3分钟的时间限制，必须在时间内消除所有图块。
2. 两个图块之间的连线不能超过两次转弯。
3. 连线路径上不能有其他未消除的图块阻挡。
4. 随着关卡提升，棋盘会变大，图块数量增加，难度提升。

操作：
点击第一个图块 - 选中该图块（会显示蓝色高亮边框）
点击第二个相同图块 - 如果可以连接，两个图块会被消除
点击已选中的图块 - 取消选择
💣 炸弹道具 - 点击屏幕上方的💣图标激活炸弹，然后点击任意图块直接消除（每关3个）

目标：
在时间限制内消除棋盘上所有的图块，完成关卡！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术，实现了流畅的图块绘制、路径检测算法和响应式布局，纯原生实现无需任何框架。
